# openvos-cfg

## 架构图

- 呼叫方向只能能接入点进，从出局点出。反方向呼叫会拒绝

```
接入点 -->> openvos -->> 出局点
```

## 概念定义

- **接入点**: 基于IP地址允许网关的呼入
    - 一个接入点对应一个IP地址
- **接入组**: 多个接入点可以聚合成一个接入组
- **出局点**: 呼叫将会送到的目的地址
- **出局组**: 多个出局点可以聚合成一个一个出局组
    - 一次呼叫失败，会尝试用组内其他出局点继续外呼

## 认证方式
- 基于接入点IP地址认证
- 基于SIP 用户名密码认证


## 支持协议

- SIP信令 TCP/UDP/TLS/WSS
- 媒体支持 RTP/SRTP
